# @grobot/templates (internal)

Runtime initialization templates used by `grobot init` and release packaging.

Current template set:

- `config.toml.example`: global home configuration sample.
- `skills/skill-creator`: built-in global `~/.grobot/skills/skill-creator` template.

Future additions can include:

- project template (`project.toml`)
- hooks/rules/skills bootstrap files
- MCP registry template
